# 남서울대학교 학우를 위한 중고 도서 판매서비스

학교에서 판매하는 전공, 비전공 도서는 너무 비쌉니다.
우리는 학우들의 도서 구매 비용의 부담을 줄이기 위해 노력합니다.

- 교통비로도 빠듯한 통학생
- 연애할 돈도 부족한 캠퍼스커플
- 술 값도 부족한 복학생
- 파릇파릇한 새내기

모두 이 서비스가 필요합니다.
